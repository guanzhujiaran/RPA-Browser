# 浏览器控制测试界面

## 使用说明

1. **启动服务器**
   ```bash
   python main.py
   ```

2. **访问测试界面**
   在浏览器中打开：`http://127.0.0.1:8000/static/test.html`

## 功能特性

### 基础配置
- **Server URL**: API服务器地址 (默认: http://127.0.0.1:8000)
- **Browser Token**: 浏览器实例的唯一标识
- **Browser ID**: 浏览器实例ID (可选)
- **无头模式**: 是否在后台运行浏览器

### 主要功能

#### 1. 页面操作
- **打开URL**: 导航到指定网页
- **截图**: 
  - 支持整页截图
  - 支持PNG/JPEG格式
  - 实时预览截图结果

#### 2. 直播功能
- **创建直播**: 开启浏览器会话直播
- **查看直播状态**: 检查直播会话信息
- **停止直播**: 结束直播会话
- **实时预览**: 在界面中直接观看直播流

#### 3. 会话管理
- **释放单个会话**: 释放指定的浏览器实例
- **释放所有会话**: 清理该Token下的所有实例

#### 4. 快速测试场景
- **测试B站页面**: 自动访问B站并截图
- **测试Google搜索**: 自动访问Google并截图
- **演示流程**: 完整的自动化操作演示

## API接口映射

| 功能 | 界面操作 | API端点 | 方法 |
|------|----------|---------|------|
| 打开URL | "打开页面"按钮 | `/browser_control/open_url` | POST |
| 截图 | "截图"按钮 | `/browser_control/screenshot` | POST |
| 创建直播 | "创建直播"按钮 | `/browser_control/live/create` | POST |
| 查看直播状态 | "查看直播状态"按钮 | `/browser_control/live/view` | GET |
| 停止直播 | "停止直播"按钮 | `/browser_control/live/stop` | POST |
| 释放会话 | "释放会话"按钮 | `/browser_control/release_session` | POST |

## 使用步骤

1. **获取Browser Token**
   - 需要先通过API创建浏览器指纹并获取token

2. **配置参数**
   - 填入有效的Browser Token
   - 可选择填写Browser ID
   - 选择是否使用无头模式

3. **执行操作**
   - 使用单独的功能按钮进行测试
   - 或使用快速测试场景进行自动化演示

## 技术特性

- **响应式设计**: 适配桌面和移动设备
- **实时反馈**: API调用结果实时显示
- **可视化预览**: 截图和直播流直接预览
- **错误处理**: 友好的错误提示信息
- **现代UI**: 使用Tailwind CSS构建美观界面

## 注意事项

- 确保FastAPI服务器正在运行
- 需要有效的Browser Token才能正常使用
- 直播功能需要浏览器实例处于活跃状态
- 建议在非生产环境中使用此测试界面
